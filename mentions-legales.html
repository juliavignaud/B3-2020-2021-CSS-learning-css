Informations générales

Je me présente, Julia Vignaud, étudiante en troisième année de Bachelor Chef de projet digital à l’ECV Digital Bordeaux. J’ai développé ce site web dans le cadre d’un projet scolaire dont le but était d’acquérir les techniques fondamentales et les nouveautés du langage CSS.

Ce projet m'a permis d'expérimenter et de mieux comprendre le fonctionnement de plusieurs propriétés CSS telles ques : Grid Layout, Flexbox Layout, Object-fit, column, float, les medias queries, les animations, les transitions, les positions...

Explications des choix techniques

La page d'accueil

La page d'accueil, présente un avant-goût des différentes pages en 4 sections, avec pour chaques section un lien (déguisé en bouton) qui redirige vers la page en question. Sur cette page on retrouve notamment l'utilisation de Grid, Flexbox ainsi que column.

Le menu

Le menu de mon site est un menu latéral responsif créé en combinant l'élément CSS ::target, les transitions et d'autres propriétés CSS (position absolute, width, height...)

La page évolutions et propriétés marquantes du CSS

La page évolution et propriétés marquantes présente 4 propriétés fondamentales du CSS : les médias queries, Flexbox, Grid et Object-fit. Sur cette page, on retrouve l'utilisation et l'expérimentation de diverses propriétés css. Pour la créations des 4 blocs, j'ai créé 4 sections différentes et j'ai utilisé Flexbox avec notamment flex-wrap : wrap; déclaré dans un média querie pour que les blocs soient responsifs. Concernant la modal qui apparaît au clic sur le lien, il n'y a aucun javascript. J'ai utilisé tout comme pour le menu, les id et l'élément CSS ::target. Globalement, on retrouve l'utilisation de transitions, filtres et animations, que ce soit pour l'animation de la modal ou l'effet de hoover sur les images.

La page personnalité

Cette page présente brièvement 10 personalités très importantes pour le langage CSS. Sur cette page, j'ai choisi d'expérimenter Flexbox et width : 100%; height: auto; pour les images. Pour cette page ainsi que pour les autres, j'ai utilisé les médias queries pour gérer l'aspect responsif du site.

La page Val Head

La page présente plus en détail Val Head en expliquant son parcours, ses actions pour le web... Sur cette page, j'ai voulu tester le fonctionnement d'object-fit : cover; sur une grande image. J'ai également mis en place une grille avec 12 colones pour pouvoir placer mes éléments où je le souhaitait sur la grille.
