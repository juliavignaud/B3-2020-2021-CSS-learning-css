Les médias queries est une recommandation officielle adaptée au W3C. Ce module de média queries CSS3 propose un rendu de page web adaptable en fonction de plusieurs facteurs tels que la résolution ou la taille de l’écran. Ces requêtes multimédias sont une des techniques de base utilisées dans la conception des sites responsive.
